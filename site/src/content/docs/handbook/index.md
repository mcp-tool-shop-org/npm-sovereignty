## What is Sovereignty?

Sovereignty is a strategy game about governance, trust, and trade. It supports offline tabletop play and optional XRPL online verification for provable game state.

This npm package is a zero-install binary wrapper. It downloads a prebuilt platform-specific binary, verifies its SHA256 checksum, and runs it locally. No Python required.
